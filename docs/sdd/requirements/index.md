# 要件定義

## 概要

ccresmonは、Claude Codeスキル（フックベース）のプラグインである。Claude Code実行中のホストPCリソース使用状況（メモリ、CPU、ディスクI/O）を監視し、負荷が高い場合に処理をブロック・警告することで、低スペックPCでも安定してClaude Codeを利用できるようにする。

### 背景・動機

- 社内の低スペックPCでClaude Codeを実行するとメモリ使用量とディスクI/Oが特に問題になる
- 身内（社内チーム）向けに個人マーケットプレイス（claudecode-plugins）で配布予定
- 公式マーケットプレイスにはPRしない

### 対象ユーザー

- 社内チームのClaude Codeユーザー（主に低スペックPC利用者）

### 技術方針

- シェルスクリプトベース（bash）
- 外部依存なし（標準Linuxコマンドのみ使用）
- Linux（WSL2含む）対応
- Claude CodeのPreToolUseフックとして動作

## 依存関係

- Claude Code（フック機能をサポートするバージョン）
- Linux環境（procfs: `/proc/loadavg`, `/proc/diskstats`）
- 標準Linuxコマンド: `free`, `cat`, `awk`, `bash`, `nproc`

## スコープ外

- macOS / Windows（ネイティブ）対応
- GUI による設定画面
- リソース使用履歴の記録・可視化
- PostToolUse（実行後）の監視
- 公式マーケットプレイスへのPR
- ネットワーク帯域の監視
- GPU使用率の監視
