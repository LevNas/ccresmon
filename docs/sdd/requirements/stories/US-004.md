# US-004: インストールとフック登録

## ユーザーストーリー

**私は** Claude Codeユーザーとして
**〜したい** ccresmonを簡単にインストールし、Claude Codeのフックとして登録したい
**なぜなら** 設定が複雑だと導入を敬遠してしまい、リソース監視の恩恵を受けられないから

## 受入基準（EARS記法）

### 機能要件

- **REQ-004-001**: システムはClaude Codeの `settings.json` に追記するフック設定の例をREADMEに記載しなければならない
- **REQ-004-002**: システムはシェルスクリプト単体で動作し、外部パッケージのインストールを必要としてはならない
- **REQ-004-003**: システムは `~/.claude/settings.json` の `hooks.PreToolUse` にコマンドとして登録できる形式でなければならない
- **REQ-004-004**: システムはフックの有効/無効を `settings.json` の編集のみで切り替えられなければならない

## 備考

- Claude Codeのhooks設定は `~/.claude/settings.json` または プロジェクトの `.claude/settings.json` に記載する
- フック登録例:
  ```json
  {
    "hooks": {
      "PreToolUse": [
        {
          "matcher": "Agent|Bash",
          "hook": "/path/to/ccresmon/ccresmon.sh"
        }
      ]
    }
  }
  ```
- ccmemoと同様のスキル形式での配布も考慮する
